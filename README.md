# 初始化珠峰博客

## 初始化项目
```
npm init -y
```

## 安装依赖的模块
```
npm install express body-parser express-session mongoose ejs multer --save
```

# 支持bootstrap
要求目录名不能大写字母、汉字以及特殊字符
```
bower init
```
新建`.bowerrc`配置文件
用来使用用bower 安装的包的安装目录
```
{
  "directory":"./public/lib"
}
```
安装bootstrap
```
bower install bootstrap --save
```
